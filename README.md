## Project Overview

This project is a basic network packet sniffer developed using Python and the Scapy library. It captures network traffic and displays useful packet information such as source and destination IP addresses, protocols, ports, and packet payloads.

## Features

- Capture live network packets
- Display source and destination IP addresses
- Identify TCP, UDP, and ICMP protocols
- Display source and destination ports
- Show packet summaries and payloads

## Technologies Used

- Python
- Scapy
- Npcap
- Visual Studio Code
